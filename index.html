AC LIGHT DIMMER USING WEB SERVER

The AC Light Dimmer using a web server is a home automation project that enables users to remotely control the brightness of an AC light through a web interface. The system incorporates a microcontroller, such as Arduino or Raspberry Pi, with a Triac-based dimmer circuit, and a Wi-Fi module to establish a connection to a local network. The microcontroller hosts a web server, allowing users to access a web page where they can interact with a slider or buttons to adjust the light intensity. This project showcases the integration of hardware and web technologies, providing a convenient and accessible solution for remote lighting control in smart homes or other IoT applications.

BRAIN TUMOR DETECTION USING CNN

Brain tumor detection using Convolutional Neural Networks (CNN) involves leveraging deep learning techniques to analyze medical images, particularly MRI scans. This approach includes training a CNN on a dataset containing both images with tumors and normal images. The trained model learns to distinguish between these images, enabling it to detect the presence of brain tumors in unseen MRI scans. This technology holds promise for assisting healthcare professionals in early and accurate diagnoses, ultimately improving patient outcomes..
